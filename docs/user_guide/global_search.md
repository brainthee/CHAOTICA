# Global Search

The search box in the top navigation bar searches across CHAOTICA from
anywhere in the app. Start typing (three characters or more) and matching
results appear grouped by type in a dropdown.

## What it searches

The global search looks across the records you have permission to see:

- **Jobs** — by title, overview, slug or Job ID
- **Phases** — by title, description or Phase ID
- **Clients**, **Billing Codes**, **Services**, **Skills**, **Qualifications**
  and **Projects** — by name/code/title
- **Members** — by name, email or alias

Results are always scoped to your permissions, so you only ever see records
belonging to organisational units and objects you are allowed to view.

## Jump straight to a Job or Phase by ID

If you already know the ID of a job or phase, you don't need to pick it from
the dropdown — type the ID and press **Enter** to go straight to it:

- **Job ID** — a plain number, e.g. `1234`
- **Phase ID** — the job/phase form, e.g. `1234-1`

If the ID matches a job or phase you can view, you are taken directly to its
page. If it doesn't match anything (or you don't have access to it), the normal
search results are shown instead.
